# P2-4 面试状态显式化 — 2026-04-29

## 问题
- `turn = chatHistory.size() / 2` 从消息条数反推阶段
- SSE 断连丢失消息会导致阶段错位
- `[SWITCH_TO_HR]` / `[AUTO_FINISH]` 标记通过字符串子串检测

## 方案
- 新增 `InterviewPhase` 枚举：OPENING / TECHNICAL / HR / CLOSING / FINISHED
- `InterviewRecord` 新增 `phase` VARCHAR(16) 字段（Flyway V2 迁移）
- `chatStream` 中显式使用 phase 驱动阶段切换
- 标记检测作为阶段切换触发信号
- turn 保留作兜底保护
