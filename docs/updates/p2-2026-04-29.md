# P2 重构记录 — 2026-04-29

## P2-2：Agent 模块决策（方案 B）

### 问题
- CoordinatorAgent、TechnicalAgent、HrAgent 三个接口已定义并注册为 Spring Bean
- 但 InterviewServiceImpl 完全绕过它们，将相同提示词作为内联字符串重复定义
- ChatConfig 中的 Agent Bean 注册是死配置

### 方案
- 删除 3 个 Agent 接口
- 删除 ChatConfig 中 Agent Bean 注册
- 提示词统一集中到 application.yml 的 interview.prompts 节点下

## P2-3：提示词外部化

### 问题
