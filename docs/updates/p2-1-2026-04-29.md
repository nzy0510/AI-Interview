# P2-1 面试编排拆分 — 2026-04-29

## 问题
- InterviewServiceImpl ~560 行，承载 5 种职责
- 会话存储、RAG 检索、评估生成与流程控制耦合

## 方案
拆为 3 个独立模块：

| 新模块 | 职责 | 接口方法 |
|--------|------|----------|
| SessionStore | 聊天消息 Redis/内存双模持久化 | load(recordId), save(recordId, msgs), delete(recordId) |
| RagRetriever | 知识原子向量检索 + 分类过滤 + 黑名单 | retrieve(position, userMessage, usedAtomIds) → List<Content> |
| EvaluationGenerator | AI 评估报告生成 + JSON 解析 + 字段映射 | generate(history, wpm, emotionJson) → InterviewRecord |

InterviewServiceImpl 缩减为 ~250 行，仅负责流程编排。
